# 🧪 Проект: автоматизированные тесты для учебного сервиса «Яндекс Самокат»

## 📋 О проекте
Проект посвящён автоматизации тестирования учебного сервиса «Яндекс Самокат»: реализованы автотесты на Selenium для ключевых сценариев (выпадающий список в блоке «Вопросы о важном», флоу оформления заказа). В работе применён паттерн Page Object, параметризация тестовых данных, кроссбраузерное тестирование (Chrome, Firefox). Проект собран как Maven‑модуль на Java 11.

## 🧪 Реализованные тесты

### 🔹 MakeOrderTest — тесты оформления заказа
- `makeOrderFromTopButton` — оформление заказа через верхнюю кнопку «Заказать»: открытие страницы, принятие куки, заполнение формы данными из параметризованного набора, проверка появления окна об успешном создании заказа.
- `makeOrderFromBottomButton` — оформление заказа через нижнюю кнопку «Заказать»: аналогичный флоу с проверкой успешного создания заказа, подтверждение работоспособности обеих точек входа.

### 🔹 QuestionsListTest — тесты блока «Вопросы о важном»
- `questionMatchAnswerTest` — проверка соответствия раскрытого ответа выбранному вопросу: скролл до блока вопросов, клик по вопросу (по индексу), получение текста ответа, сравнение с ожидаемым значением из параметризованных данных (покрыто 8 вопросов).
